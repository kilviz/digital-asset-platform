## Запуск

Устанавливаем необходимые зависимости. В корневой папке запускаем команду:

```
npm install
```
Запускаем тесты:

```
npx hardhat test
```

Запускаем блокчейн:
```
npx hardhat node
```
Деплоим смарт контракты:

```
npx hardhat run scripts/deploy.js --network localhost
```
